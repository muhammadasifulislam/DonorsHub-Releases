**DonorsHub** is a highly optimized, zero-cost, modern blood donation ecosystem designed for Bangladesh. It connects blood seekers with the nearest available donors instantly using GPS and smart routing.

## 🚀 Features
- **Smart GPS Sorting:** Automatically finds the nearest available donors using real-time distance calculations.
- **Zero-Cost SMS Alert:** Sends bulk emergency SMS to available donors without any external API costs.
- **Push Notifications & Emails:** Automated alerts via Vercel Backend and Firebase Cloud Messaging.
- **Multi-language Support:** Instantly toggle between English and Bengali (বাংলা).
- **Thalassemia & Rare Cases:** Dedicated registry for Thalassemia patients.
- **Modern UX:** Soft UI, interactive blood dials, and animated real-time counters.
